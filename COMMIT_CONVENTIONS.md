# Commit Conventions

Use Conventional Commits for this repository.

Format:

`type(scope): short summary`

Examples:

* `feat(summary): add PR summarization route`
* `fix(prompt): make summarization prompt deterministic`
* `refactor(client): simplify OpenAI service wrapper`
* `docs(readme): update local setup steps`
* `chore(deps): bump OpenAI SDK`
* `test(ai): add summary response tests`

Recommended types:

* `feat`
* `fix`
* `refactor`
* `docs`
* `chore`
* `test`
* `ci`
* `build`
